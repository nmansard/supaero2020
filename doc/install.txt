The following dependencies are needed:

- for using the pendulum models, you need Pinocchio (Python 2.7 only). To be install using APT on Ubuntu 18.04 or follow https://stack-of-tasks.github.io/pinocchio/download.html.

- You also need a bunch of other smaller packages around pinocchio. Once pinocchio is there, apt robotpkg-py36-pinocchio robotpkg-py36-qt4-gepetto-viewer-corba robotpkg-example-robot-data robotpkg-osg-dae

- Finally, we need some standard Python packages (also on apt): ipython3 python3-pip freeglut3 python3-matplotlib python3-pil.imagetk jupyter python3-scipy python3-matplotlib jupyter python3-scipy python3-numpy
